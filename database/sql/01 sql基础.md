# Sql

## 基本语法

- sql语句末尾可以有分号，也可以没有分号

- DDL数据定义
    - CREATE DATABASE
    - ALTER DATABASE
    - CREATE TABLE
    - ALTER TABLE
    - DROP TABLE
    - CREATE INDEX
    - DROP INDEX

- DML数据操作语言
    - SELECT
    - UPDATE
    - DELETE
    - INSERT INTO

## SELECT

```sql
SELECT 列名称 FROM 表名称
SELECT * FROM 表名称
SELECT DISTINCT 列名称 FROM 表名称
```
